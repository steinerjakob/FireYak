FireYak helps fire departments quickly find the nearest usable water source — fire hydrants, suction points, water tanks, fire water ponds and fire stations on an interactive map.

All data comes from the community-driven project OpenStreetMap.

<b>Find water sources nearby</b>
- Displays fire hydrants, suction points, water tanks, fire water ponds and fire stations near you
- GPS-based location: see the nearest water sources at a glance
- Distance display in meters and kilometers
- Estimated number of supply hoses

<b>Detailed information</b>
- Hydrant type (pillar, underground, wall)
- Pipe diameter, pressure, flow rate
- Couplings (type, diameters)
- Water source (main, pond, stream, river, lake, tank, well)
- Capacity, reference number, operator, address
- Coordinates and OSM ID

<b>Edit water sources</b>
- Add new hydrants, suction points and water tanks directly in the app
- Edit or delete existing entries
- All changes sync to OpenStreetMap in real time
- OpenStreetMap login

<b>Relay pump / supply line calculator</b>
- Set fire object (target) and suction point (water source)
- Optional waypoints for routing
- 3D hose distance calculation using elevation data
- Number and position of intermediate pumps
- Pressure at each pump and at the fire object
- Adjustable parameters: output pressure, minimum input pressure

<b>More features</b>
- Start navigation to any water source
- View photos from Wikimedia Commons
- Share a link to any water source
- Open directly on openstreetmap.org or edit in OSM editor
- Dark mode
- Available in English and German
- Completely free and open source
